# Knowledge Search

The Knowledge Search tool provides advanced semantic search capabilities across transport accident investigation reports.

## Features

### Semantic Search

Unlike traditional keyword search, semantic search understands the meaning and context of your query, finding relevant documents even when they don't contain your exact words.

### Filtering Options

Filter your search results by:

- **Document Type**: Reports, safety recommendations, etc.
- **Agency**: TAIC, ATSB, TSB
- **Year Range**: Filter by publication date
- **Transport Mode**: Aviation, rail, marine

### Analytics and Visualization

View search results through:

- Result tables with key metadata
- Visualizations showing patterns across results
- Distribution of document types and years to identify trends
- Downloadable graphs (click the camera icon in the top right of each graph)
- Export capabilities for further analysis

## Using Knowledge Search

### Basic Search

1. Navigate to the Knowledge Search tab
2. Enter your search query in the search box
3. Click Search
4. Review results in the table below

### Example Searches

```
fatigue-related accidents in aviation
```

```
safety recommendations about railway level crossings
```

```
findings from marine collision investigations
```

### Applying Filters

#### Document Type Filter

Select one or more document types:

- Investigation Reports
- Safety Recommendations
- Interim Reports
- Occurrence Reports

#### Agency Filter

Choose which agencies to include:

- TAIC (New Zealand)
- ATSB (Australia)
- TSB (Canada)

#### Year Range

Use the slider to select a specific time period:

- Drag the handles to adjust start and end years
- Results update automatically

#### Transport Mode

Filter by transport sector:

- Aviation
- Rail
- Marine

### Relevance Filters

When performing a smart search, results are ranked by relevance (1 is most relevant, 0 is least relevant). The default relevance filter is 0.2 to filter out irrelevant results. You can adjust this if you're not getting enough results or want more. The filter can be sensitive, so try adjusting in increments of 0.05.

### Understanding Results

Each result shows:

- **Document Title**: Name of the report or document
- **Agency**: Organization that published it
- **Year**: Publication year
- **Mode**: Transport sector
- **Relevance Score**: How well it matches your query
- **Link**: Direct link to view the full document

### Analytics

Click the **"Results graph"** dropdown just above the results to view:

- Distribution of results by year
- Breakdown by transport mode
- Agency representation
- Document type distribution

Each graph can be downloaded by clicking the camera icon in the top right of the graph.

### Post-Search Filtering

After performing a search, you can further refine your results using post-search filtering. In the search bar at the top right of the results table, type words to perform an exact text filter on the results already returned by the smart search. This is useful for narrowing down results to specific topics or keywords.

### Regular Text Search

You can also do a regular text search by enclosing your search terms in double quotes. For example: `"fuel starvation"`. This will return all results that contain that exact text regardless of the meaning.

### Exporting Results

Export your search results:

1. Click "Export Results"
2. Choose format (CSV or Excel)
3. File downloads automatically

Exported data includes all metadata and relevance scores.

## Advanced Search Tips

!!! tip "Improving Search Quality"
    - Use descriptive phrases rather than single keywords
    - Include context about what you're looking for
    - Try different phrasings if results aren't relevant
    - Combine search with filters for more precise results

!!! example "Search Examples"
    **Broad topic**: "runway safety incidents"
    
    **Specific aspect**: "contributing factors to runway incursions at controlled airports"
    
    **Safety focus**: "recommendations for improving fatigue management in commercial aviation"

## Search Limitations

- Results are limited to documents in the knowledge base
- Very recent publications may not yet be indexed
- Relevance scores are relative to your specific query
- Some specialized terminology may require specific phrasing

## Troubleshooting

**No results found?**

- Try broader search terms
- Remove some filters
- Check for typos
- Try alternative phrasing

**Too many irrelevant results?**

- Be more specific in your query
- Apply additional filters
- Focus on specific aspects of your topic
