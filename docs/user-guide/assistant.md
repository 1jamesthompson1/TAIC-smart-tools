# Smart Assistant

The Smart Assistant is an AI-powered tool that helps investigators and researchers by providing intelligent, context-aware responses.

## Features

### Conversational Interface

The assistant maintains context throughout your conversation, allowing you to:

- Ask follow-up questions
- Refine previous queries
- Build on earlier responses

### Knowledge Base Access

The assistant can search through:

- TAIC investigation reports and safety recommendations
- ATSB (Australian Transport Safety Bureau) reports
- TSB (Transportation Safety Board of Canada) reports

### Report Reading

The assistant can:

- Retrieve and read specific reports
- Analyze report content
- Extract relevant information
- Provide summaries

## Using the Assistant

### Starting a Conversation

1. Navigate to the Smart Assistant tab
2. Type your question in the chat input
3. Press Enter or click Send
4. Wait for the assistant to respond

### Example Queries

```
What were the contributing factors in marine accidents involving fishing vessels in 2020?
```

```
Can you summarize report AO-2023-001?
```

```
What safety recommendations have been made regarding fatigue in rail operations?
```

### Conversation Management

- **New Conversation**: Start fresh with a new conversation
- **Conversation History**: View and resume previous conversations
- **Delete Conversations**: Remove conversations you no longer need

### Best Practices

!!! tip "Getting Better Results"
    - Be specific in your questions
    - Provide context when needed
    - Use follow-up questions to refine results
    - Reference specific document types or time periods

!!! warning "Limitations"
    - The assistant's knowledge is based on the documents in the database
    - Responses are AI-generated and should be verified for critical decisions
    - The assistant cannot access documents outside its knowledge base

## Advanced Features

### Tool Usage

The assistant can use multiple tools:

- **Search Tool**: Finds relevant documents
- **Read Report Tool**: Retrieves full report content
- **Reasoning Tool**: Performs complex analytical thinking

The assistant automatically selects appropriate tools based on your query.

### Response Formatting

Responses include:

- Inline citations to source documents
- Structured information when appropriate
- Links to referenced reports

## Troubleshooting

**Assistant not responding?**

- Check your internet connection
- Refresh the page
- Start a new conversation

**Unexpected responses?**

- Rephrase your question
- Provide more context
- Break complex queries into simpler questions
