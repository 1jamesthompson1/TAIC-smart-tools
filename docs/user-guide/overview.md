# User Guide Overview

This guide will help you get started with TAIC Smart Tools.

[Back to smart tools](/tools){ .md-button }

## Available Tools

### Smart Assistant

The Smart Assistant is an AI-powered conversational interface that can:

- Answer questions about transport accident investigation
- Search through TAIC, ATSB, and TSB documents
- Read and analyze specific reports
- Maintain conversation context across multiple interactions

[Learn more about the Smart Assistant →](assistant.md)

### Knowledge Search

The Knowledge Search tool provides:

- Semantic search across investigation reports
- Advanced filtering by document type, agency, year, and mode
- Visual analytics and insights
- Export capabilities for search results

[Learn more about Knowledge Search →](search.md)

### Underlying Database

Both tools search across a database of investigation reports from TAIC, ATSB, and TSB — including report sections, safety issues, recommendations, and summaries.

[Learn about the database →](database.md)

## Getting Started

1. **Access the Application**: Navigate to your organization's deployment URL
2. **Authenticate**: Log in using your Azure AD credentials
3. **Choose a Tool**: Select either the Smart Assistant or Knowledge Search
4. **Start Exploring**: Begin asking questions or searching documents

## System Requirements

The application works best with:

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Stable internet connection
- Screen resolution of 1280x720 or higher
