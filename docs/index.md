# TAIC Smart Tools Documentation

Welcome to the documentation for TAIC Smart Tools - a platform providing AI-powered tools for the New Zealand Transport Accident Investigation Commission.

[To Smart Tools →](/tools){ .md-button .md-button--primary }

## Overview

TAIC Smart Tools consists of two main features:

🤖 **Smart Assistant** - An interactive AI assistant that helps investigators and researchers by providing context-aware responses using TAIC's knowledge base.

🔍 **Knowledge Search** - Advanced semantic search capabilities across TAIC, ATSB, and TSB documents with filtering and analytics.

## Features

- **AI-Powered Conversations**: Context-aware chat interface with memory
- **Vector Search**: Semantic search across thousands of investigation reports
- **Advanced Filtering**: Filter by document type, agency, year, and transport mode
- **Analytics**: Visual insights and export capabilities
- **Secure**: OAuth authentication with Azure AD
